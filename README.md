# About

This app allows you to make, edit, search and filter notes very fast. I've been trying to create a note-taking app that will have only crucial features. It has tags, to-do lists, and other useful tools to format your notes as you wish.

## Link to app

https://danyanikora.github.io/notes/

## Stack

- [React](https://reactjs.org/)
- [Redux Toolkit](https://redux-toolkit.js.org/)
- [Parcel](https://parceljs.org/)
- [Quill](https://quilljs.com/)
